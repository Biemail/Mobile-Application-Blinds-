# D'Decor Blinds Mobile Application

## Overview
D'Decor Blinds is an official mobile application developed for authorised dealers and internal staff to access product information and manage daily business operations efficiently.

The application is built using **.NET MAUI** and is available on the **Google Play Store**.

---

## Key Features
- Access complete and updated E-Catalogues
- Track customer orders with real-time status
- Check product stock availability
- Enhanced dashboard with quick access to key services
- Clean, user-friendly interface optimised for business use
- Secure access for authorised users only

---

## Technology Stack
- **Framework:** .NET MAUI
- **Language:** C#
- **UI:** XAML
- **Platform:** Android
- **Architecture:** Shell-based navigation
- **Storage:** Preferences (for local session data)

---

## Application Modules
- Dashboard
- Stock Check
- Order Tracking
- E-Catalogue
- Account Management (Profile, Help, Logout)

---

## Installation & Setup

### Prerequisites
- Visual Studio 2022 (or later)
- .NET MAUI workload installed
- Android SDK
- Physical Android device or emulator
